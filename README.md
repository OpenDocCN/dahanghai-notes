## 下载

### Docker

```
docker pull apachecn0/dahanghai-notes
docker run -tid -p <port>:80 apachecn0/dahanghai-notes
# 访问 http://localhost:{port} 查看文档
```

### NPM

```
npm install -g dahanghai-notes
dahanghai-notes <port>
# 访问 http://localhost:{port} 查看文档
```
